Scala Space Game

Overview

This project was meant to imitate the classic arcade game of Galaga, but with an added functionality of rewinding time. This is achieved by using a Scala/JavaFX libraries and some custom vector datatypes and GameStates to store information on a frame-by-frame basis, then preserving a Stack of GameStates and popping the contents of the Stack off when the rewinding functionality is triggered. Additional functionality includes scoring based on eliminating enemies and firing enemy bullets out of the air, collisions with enemies, accelerating enemy fire rate and movement speed by round and elimination count, and window flag management. I layed out framework to add powerups as well, including a rapid fire powerup and a teleportation powerup.

You can explore the source code or try it for yourself at Scala Space Game.
